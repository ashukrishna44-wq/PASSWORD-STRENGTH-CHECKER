# PASSWORD-STRENGTH-CHECKER
simple python script to check password strength whether the password is strong, weak, or average based on:
length >= 8
lowercase letters
uppercase letters
special characters
digits

#why this matters
weak passwords are one of the biggest cybersecurity risks. this small tool helps users to undestand what makes a password stronger.
